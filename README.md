# react-image

## Getting Started

npm install react-image-to-lazyload -D

yarn add react-image-to-lazyload -D

Example:

```js
import * as React from 'react';
import { Image } from 'react-image-to-lazyload';

const ImageList = function () {
	return <Image src="./static/img.jpg"></Image>;
};
```
